GUIDA ALL'USO CORRETTO DI SHAREBOT
----------------------------------------------------------------------------------------------
Sviluppatore: https://github.com/FebVeg
----------------------------------------------------------------------------------------------
IMPORTANTE:
\ Quando il programma condividerà il Post sui tuoi gruppi, EVITA DI SCHIACCIARE ALTRI PULSANTI: Facendo cosi, disturberai il processo di sistema del Computer (Windows o Linux, è lo stesso)
 \ Le tempistiche di condivisione e degli input automatici sono RANDOMICHE
  \ L'eseguibile .exe per windows, potrebbe risultato come FALSO POSITIVO dagli antivirus. In caso l'antivirus rilevasse il programma come potenzialmente non desiderato, potete percorrere due strade:
   \ Prima strada: Dire all'antivirus che il programma è legittimo e quindi di creare una eccezzione per ShareBOT
   \ Seconda strada: Installare Python3 ultima release (o ultimo rilascio) ed eseguirlo semplicemente con "Doppio Click"
----------------------------------------------------------------------------------------------
BOX > LOGIN
Informazioni: Compila i campi 'Email', 'Password'.

- Email + Pass
\ In questi campi bisogna inserire le proprie credenziali di Facebook (Sicurezza: ShareBOT non salva le credenziali sul computer, nè le condividerà su internet)

- LOGOUT (Non serve compilare nessun campo, serve solo avere il driver attivo per effettuare la disconnessione dell'utenza, altrimenti: se si cercherà di disconnettersi senza il driver in funzione, si otterrà un errore)
\ Sicurezza dei dati di accesso: Il driver verrà avviato in una modalità chiamata "INCOGNITO MODE", che sta a significare che ogni cosa succeda nel driver (Input dell'utente, Chiavi di accesso, Altro..) non verrà salvata su disco.

- INFORMAZIONI
\ Stamperanno nella console questo tutorial

- SVILUPPATORE
\ Aprirà il browser direttamente sulla mia pagina Github
----------------------------------------------------------------------------------------------
BOX > Lista Gruppi
Informazioni: La lista gruppi è un file di testo, per ottenerlo leggi quanto segue.
- File: [___________] Apri
\ Questo input ti chiede di aprire il file della lista gruppi, si aprirà l'esplora risorse (Windows) o il file manager di Linux (tipo: caja, thunar, ecc..)

- Azione 'ESPORTA'
\ Una volta che hai eseguito il Login dal programma, potrai esportare tutti i gruppi di cui fai parte in un file di testo.
 \ Il file verrà salvato nello stesso percorso del programma, ovvero nella stessa cartella del programma.
  \ Il file esportato si chiamerà: 'ListaGruppi.txt' e verrà importato automaticamente nel programma.

- Azione 'MODIFICA' 
Informazioni: Decidere a chi condividere il post modificando il file di testo esportato.
\ Questa funzione è in grado di aprire il file 'ListaGruppi.txt' dal tuo editor di testo predefinito (per Windows è il blocconote).
 \ Una volta aperto, controlla i gruppi. Avrai il link sulla sinistra e il nome sulla destra.
  \ Se noti il nome di un gruppo di cui non vuoi condividere il post, elimina TUTTA la riga del gruppo (E' possibile fare questa modifica su più i gruppi).
   \ Quando avrai finito di modificare la lista gruppi, salva il file e chiudilo.
----------------------------------------------------------------------------------------------
BOX > Messaggio/Post
Informazioni: Questa è l'area di testo dove andrai a scrivere il post da condividere
\ Non ci sono limiti di caratteri.
----------------------------------------------------------------------------------------------
BOX > Prompt dei comandi (Console)
Informazioni: La console è uno strumento che ti mette al corrente su quello che sta facendo ShareBOT
\ E' importante prestare attenzione su quello che succede al programma.
 \ Si possono manifestare degli errori o dei problemi.
  \ In caso si verificassero errori, ti prego di inviarmi una mail a 'febveg_sharebot_issues@hi2.in' con OGGETTO 'Problema ShareBOT' e in allegato la foto dell'errore.
----------------------------------------------------------------------------------------------
BOX > PULSANTI
Informazioni: Questi tre pulsanti sono dei "gestori" del programma.
- CONDIVIDI
\ Questo pulsante va premuto SOLO dopo aver eseguito il Login, se no restituirà un errore.
 \ Se avrai eseguito il Login e scritto il Post da condividere, potrai anche non importare nel programma la lista gruppi, perchè 'Avvia' riconosce se la lista gruppi è stata importata oppure no. Se non è stata importata, allora ShareBOT condividerà il Post su TUTTI i gruppi di cui fai parte, in automatico.

- Esci
\ Questo pulsante DEVE ESSERE SEMPRE USATO QUANDO SI CHIUDE IL PROGRAMMA!
 \ Ha una funzione tutta sua incorporata, che tenta di chiudere il programma in maniera logica per il sistema senza causare crash improvvisi.
